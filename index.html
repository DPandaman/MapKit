Welcome to MapKit

MapKit is a collection of ready-to-use templates designed to help developers build projects faster with the Google Maps API. Whether you're exploring a new idea or launching a full product, MapKit lets you skip the setup and get straight to building. Best part? We're built by the community, for the community.

Browse Templates
⚠️ Previews Disabled: For security reasons, we’ve removed our Google Maps API key from the site. To preview any template, please download the project files, create a .env file with your own Google Maps API key, and run it locally. See more about getting your own API key and keeping it secure on the Getting Started page and the files on the Github.

We’re currently working on publishing the full site with secure server-side API key handling, allowing users to preview templates directly in the browser. Unfortunately, we couldn’t implement this before the deadline, but it’s a top priority going forward. Firebase integration is also in progress to support features like template hosting and user authentication for uploads from the community in the near future.

We apologize for the inconvience and hope to make MapKit a great resource for Googlers around the world!
